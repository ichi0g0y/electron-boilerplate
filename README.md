# electron-boilerplate

## 注意 - Caution

このプロジェクトのコードは粗雑です、もし使用する場合はご自身でよしなにしてください。
The code in this project is crude, so if you use it, format it yourself and use it.

ワーニングとか出てるかもしれないです。

## 概要 - Overview

- typescript
- electron
- react + redux
- semantic-ui-react with theme
- emotion
- webpack
- no babel
- no redux-thunk

## 使い方 - Start

```zsh
yarn
yarn build:theme
yarn dev
```
